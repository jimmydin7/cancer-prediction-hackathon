Cancer Prediction Using Machine Learning

this is cancer prediction project in python. I made it to raise awaireness for cancer because I lost a relative to this stupid disease. It uses machine learning (random forest classifier) and a public kaggle dataset. Users just enter some medical data and history, and they get a diagnosis (chance they have cancer)

How It Works

a random forest model analyzes multiple patient factors to assess cancer risk probability

Data Collection

Patient data including demographics, lifestyle factors, and medical history is collected and preprocessed

Model Training

Random Forest algorithm trains on 80% of the data, building multiple decision trees for robust predictions

Risk Assessment

The model outputs probability scores indicating cancer risk levels based on input patient features

Patient Features Analyzed

The model considers eight key factors from the cancer_db.csv dataset

Age

Patient age demographic

Gender

Biological sex factor

BMI

Body Mass Index measurement

Smoking

Smoking status and history

Genetic Risk

Hereditary predisposition factors

Physical Activity

Exercise and lifestyle patterns

Alcohol Intake

Consumption frequency and amount

Cancer History

Previous cancer diagnosis record

Why Random Forest?

Advantages

  • • Excellent balance of accuracy and interpretability
  • • Handles mixed data types effectively
  • • Reduces overfitting through ensemble averaging
  • • Provides feature importance rankings

Implementation

  • • Built with scikit-learn RandomForestClassifier
  • • Uses train_test_split for data validation
  • • Returns probability scores via predict_proba
  • • Reproducible results with random_state=42
